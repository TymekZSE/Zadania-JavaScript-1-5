1) Kiedy powstał JavaScript?

JavaScript powstał w 1995 roku.

2) Dlaczego jest to język skryptowy?

JavaScript jest językiem skryptowym, ponieważ jest interpretowany bezpośrednio przez przeglądarkę, więc nie musi być kompilowany, by można było zobaczyć efekty jego działania.

3) Gdzie jest wykonywany JS (klient lub serwer)?

JavaScript jest wykonywany po stronie klienta, choć może być także używany na serwerze (np. Node.js).

4) Dlaczego JS to bezpieczne narzędzie?

JavaScript jest bezpieczny, ponieważ działa w izolowanym środowisku przeglądarki, które ogranicza jego dostęp do zasobów komputera użytkownika, minimalizując ryzyko niepożądanych działań.

5) Czy jest to język obiektowy, jeśli tak to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ umożliwia tworzenie obiektów, definiowanie metod, właściwości oraz zdarzeń, co jest cechą programowania obiektowego.

6) Wymień zalety JS:

Zaleta1: Prosty w nauce – można go zacząć używać po krótkim czasie nauki.

Zaleta2: Bardzo popularny język skryptowy z ogromną ilością dokumentacji i przykładów.

Zaleta3: Odciąża serwery, umożliwiając wykonywanie kodu po stronie użytkownika.

7) Framework to:

Framework to zestaw narzędzi, bibliotek i zasad, które ułatwiają tworzenie aplikacji przez dostarczanie gotowych rozwiązań i struktury kodu.

8) Do aplikacji mobilnych można użyć Framework:

Do aplikacji mobilnych można użyć frameworków takich jak React Native, które pozwalają na tworzenie aplikacji na różne systemy operacyjne jednocześnie.

9) Aplikacje responsywne to:

Aplikacje responsywne to aplikacje dostosowujące się do różnych rozmiarów ekranu, zapewniając prawidłowe wyświetlanie na urządzeniach mobilnych i komputerach.

10) Wyjaśnienie kodu:

‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›

To przycisk, który po kliknięciu wywołuje funkcję WinOpen z argumentem, by otworzyć nowe okno lub wykonać inne działanie.

11) Wyjaśnienie kodu:

window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

To polecenie otwiera nowe okno przeglądarki z określonymi parametrami:

  • toolbar=no: wyłącza pasek narzędzi,
  • directories=no: ukrywa pasek katalogów,
  • menubar=no: ukrywa menu,
  • height=280: ustawia wysokość okna na 280 pikseli,
  • width=160: ustawia szerokość okna na 160 pikseli,
  • top=200: pozycjonuje okno 200 pikseli od góry,
  • left=200: pozycjonuje okno 200 pikseli od lewej krawędzi.

12) Opis instrukcji window.close():

Instrukcja window.close() zamyka bieżące okno lub kartę przeglądarki, w której została wywołana, co przydaje się po zakończeniu określonych działań.
